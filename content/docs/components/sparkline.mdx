---
title: Sparkline
description: SketchSparkline — a hand-drawn inline trend line for a series of values.
---

`SketchSparkline` plots a sequence of numbers as a rough.js polyline. It's
fluid — it fills the width of its container, so wrap it to size it.

## Default

<ComponentPreview
  code={`import { SketchSparkline } from "blackchalk";

<SketchSparkline data={[3, 7, 4, 9, 6, 11, 8, 14, 10]} />`}
>
  <div style={{ width: 420, maxWidth: "100%" }}>
    <SketchSparkline data={[3, 7, 4, 9, 6, 11, 8, 14, 10]} />
  </div>
</ComponentPreview>

## Props

| Prop     | Type       | Default |
| -------- | ---------- | ------- |
| `data`   | `number[]` | —       |
| `height` | `number`   | `30`    |
