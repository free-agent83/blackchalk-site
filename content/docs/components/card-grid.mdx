---
title: Card Grid
description: SketchCardGrid — a responsive rough-styled grid for laying out SketchCard items.
---

`SketchCardGrid` arranges `SketchCard` children in a uniform column grid using
`SketchGrid` under the hood. It's fluid — it fills its container, so wrap it to
control width.

## Props

| Prop       | Type        | Default |
| ---------- | ----------- | ------- |
| `columns`  | `number`    | `3`     |
| `gap`      | `number`    | `24`    |
| `children` | `ReactNode` | —       |
