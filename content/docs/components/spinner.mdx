---
title: Spinner
description: SketchSpinner — a hand-drawn loading spinner.
---

`SketchSpinner` draws a rough.js spinner that rotates while `spinning` is true — a
low-fidelity "working on it" cue that matches the rest of the sketch.

## Default

<ComponentPreview
  code={`import { SketchSpinner } from "blackchalk";

<SketchSpinner />`}
>
  <SketchSpinner />
</ComponentPreview>

## Props

| Prop        | Type            | Default |
| ----------- | --------------- | ------- |
| `spinning`  | `boolean`       | `true`  |
| `size`      | `number`        | —       |
| `duration`  | `number`        | —       |
| `className` | `string`        | —       |
| `style`     | `CSSProperties` | —       |
