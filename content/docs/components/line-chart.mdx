---
title: Line Chart
description: SketchLineChart — a hand-drawn line chart with sketched axes and data points.
---

`SketchLineChart` plots a series of labelled values as a rough.js line with sketched axes
and filled data-point circles. It's fluid — it fills the width of its container, so wrap
it to size it. Pass `area` to shade the region under the line.

## Default

  code={`import { SketchLineChart } from "blackchalk";

<SketchLineChart

    { label: "Mon", value: 12 },
    { label: "Tue", value: 28 },

    { label: "Thu", value: 44 },
    { label: "Fri", value: 33 },
    { label: "Sat", value: 51 },
  ]}
/>`}

    <SketchLineChart

        { label: "Fri", value: 33 },

</ComponentPreview>

## Props

| Prop     | Type                                   | Default |
| -------- | -------------------------------------- | ------- |
| `data`   | `{ label: string; value: number }[]`   | —       |
| `height` | `number`                               | `250`   |
| `area`   | `boolean`                              | `false` |
