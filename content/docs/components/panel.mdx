---
title: Panel
description: SketchPanel — a titled card for grouping related content.
---

`SketchPanel` wraps content in a rough.js bordered card with an optional heading —
a low-fidelity container for grouping related items on a page.

## Default

<ComponentPreview
  code={`import { SketchPanel } from "blackchalk";

<SketchPanel title="Account details">
  <p style={{ margin: 0 }}>
    A panel is a titled card for grouping related content on a page.
  </p>
</SketchPanel>`}
>
  <div style={{ width: 420, maxWidth: "100%" }}>
    <SketchPanel title="Account details">
      <p style={{ margin: 0 }}>
        A panel is a titled card for grouping related content on a page.
      </p>
    </SketchPanel>
  </div>
</ComponentPreview>
