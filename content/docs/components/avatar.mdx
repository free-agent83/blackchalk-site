---
title: Avatar
description: SketchAvatar — a hand-drawn initials avatar with a rough.js circle border.
---

`SketchAvatar` renders up to two initials inside a sketched circle — a low-fidelity
identity placeholder that fits naturally alongside other hand-drawn components.

## Default

<ComponentPreview
  code={`import { SketchAvatar } from "blackchalk";

<SketchAvatar initials="AL" />`}
>
  <SketchAvatar initials="AL" />
</ComponentPreview>

## Group

<ComponentPreview
  code={`import { SketchAvatar } from "blackchalk";

<div style={{ display: "flex", gap: 8 }}>

  <SketchAvatar initials="GH" size={64} />

## Props

| Prop       | Type     | Default |
| ---------- | -------- | ------- |
| `initials` | `string` | —       |
| `size`     | `number` | `40`    |
