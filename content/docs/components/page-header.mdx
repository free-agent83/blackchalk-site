---
title: Page Header
description: SketchPageHeader — a hand-drawn page title block with optional description and actions.
---

`SketchPageHeader` renders a page-level heading, an optional description, and an
optional actions slot, separated from the page body by a sketched divider.

## Default

<ComponentPreview
  code={`import { SketchPageHeader } from "blackchalk";

<SketchPageHeader title="Projects" />`}
>
  <div style={{ width: 480, maxWidth: "100%" }}>
    <SketchPageHeader title="Projects" />
  </div>
</ComponentPreview>

## With description and actions

<ComponentPreview
  code={`import { SketchPageHeader } from "blackchalk";
import { SketchButton } from "blackchalk";

<SketchPageHeader
  title="Team members"
  description="Manage who has access to this workspace."
  actions={<SketchButton>Invite</SketchButton>}
/>`}
>
  <div style={{ width: 480, maxWidth: "100%" }}>
    <SketchPageHeader
      title="Team members"
      description="Manage who has access to this workspace."
      actions={<SketchButton>Invite</SketchButton>}
    />
  </div>
</ComponentPreview>
