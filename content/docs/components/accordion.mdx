`SketchAccordion` renders a list of collapsible panels separated by rough.js
dividers. The chevron rotates on open, height animates with a spring, and a
single panel is open at a time by default.

## Default

## Multiple open

Pass `allowMultiple` to let more than one panel stay open at the same time.

<ComponentPreview
  code={`<SketchAccordion
  items={items}
  allowMultiple
  defaultOpen={[0, 1]}
/>`}
>
  <div style={{ width: 420, maxWidth: "100%" }}>
    <SketchAccordion
      items={[
        { title: "What is Blackchalk?", content: "A hand-drawn, monochrome React UI kit built on rough.js." },
        { title: "Is it responsive?", content: "Yes ‚Äî components collapse and reflow on small screens." },
        { title: "Can I theme it?", content: "Light and dark, derived from a single ink token." },
      ]}
      allowMultiple
      defaultOpen={[0, 1]}
    />
  </div>
</ComponentPreview>
