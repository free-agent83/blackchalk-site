---
title: Cart Item
description: SketchCartItem — a hand-drawn cart row with quantity controls and an optional remove button.
---

`SketchCartItem` renders a single line item for a shopping cart — a sketch image
placeholder, product name, price, and ± quantity stepper. Pass `onRemove` to show a
close button on the right edge.

## Default

<ComponentPreview
  code={`import { SketchCartItem } from "blackchalk";

<SketchCartItem
  name="Hand-Knit Wool Beanie"
  price="24.00"
  quantity={2}
/>`}
>
  <div style={{ width: 420, maxWidth: "100%" }}>
    <SketchCartItem
      name="Hand-Knit Wool Beanie"
      price="24.00"
      quantity={2}
    />
  </div>
</ComponentPreview>

Pass `onRemove` to surface a close button on the right edge, and `onQuantityChange`
to handle the ± stepper.

## Props

| Prop               | Type                        | Default |
| ------------------ | --------------------------- | ------- |
| `name`             | `string`                    | —       |
| `price`            | `string`                    | —       |
| `quantity`         | `number`                    | —       |
| `onQuantityChange` | `(qty: number) => void`     | —       |
| `onRemove`         | `() => void`                | —       |
