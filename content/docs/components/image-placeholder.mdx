---
title: Image Placeholder
description: SketchImagePlaceholder — a hand-drawn image placeholder with a sketchy frame and icon.
---

`SketchImagePlaceholder` draws a rough.js framed rectangle with a sketchy mountain-and-sun icon
inside — a low-fidelity stand-in for images that haven't loaded or been set yet. Pass a fixed
`width` for thumbnails, or omit it to fill the parent container.

## Default

<ComponentPreview
  code={`import { SketchImagePlaceholder } from "blackchalk";

<SketchImagePlaceholder width={240} height={150} />`}
>

</ComponentPreview>

## Props

| Prop        | Type                  | Default |
| ----------- | --------------------- | ------- |
| `width`     | `number \| string`    | `"100%"` |
| `height`    | `number`              | `150`   |
| `label`     | `string`              | —       |
| `className` | `string`              | —       |
| `style`     | `CSSProperties`       | —       |
