---
title: Toggle
description: SketchToggle — a hand-drawn on/off switch with animated knob.
---

`SketchToggle` renders a rough.js pill track with a spring-animated knob. It
works uncontrolled (flips its own state) or fully controlled when you pass
`onChange`.

## Default

<ComponentPreview
  code={`import { SketchToggle } from "blackchalk";

<SketchToggle checked={false} aria-label="Toggle" />`}
>
  <SketchToggle checked={false} aria-label="Toggle" />
</ComponentPreview>

## With label

<ComponentPreview
  code={`import { SketchToggle } from "blackchalk";
import { useState } from "react";

export function Demo() {
  const [on, setOn] = useState(false);
  return <SketchToggle checked={on} onChange={setOn} label="Dark mode" />;
}`}
>
  <SketchToggle checked={false} label="Dark mode" />
</ComponentPreview>
